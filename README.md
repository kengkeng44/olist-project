\# Olist E-Commerce Data Analysis



巴西電商平台 Olist 的資料分析專案，使用 SQL 進行資料查詢，Python 處理與視覺化，最後以 Tableau 製作儀表板。



\## 工具



Python / SQLite / DBeaver / Jupyter Notebook / Tableau



\## 分析內容



\- 月營收趨勢（2017–2018）

\- 熱銷商品類別 Top 10

\- 客戶評分分布

\- 各州總營收 Top 10

\- 各州物流效率：預期 vs 實際送達天數
