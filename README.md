\# Olist E-Commerce Data Analysis



使用巴西電商平台 Olist 的公開資料集，透過 SQL 查詢與 Python 視覺化進行資料分析。



\## 工具

\- Python（pandas、matplotlib）

\- SQLite / DBeaver

\- Jupyter Notebook



\## 分析主題



1\. \*\*月營收趨勢\*\*：2017–2018 年整體營收成長明顯，2018 年營收較 2017 年顯著提升

2\. \*\*熱銷商品類別 Top 10\*\*：bed\_bath\_table、health\_beauty 等類別貢獻最高營收

3\. \*\*客戶評分分布\*\*：超過半數訂單獲得 5 星評價，整體滿意度偏高

4\. \*\*各州總營收 Top 10\*\*：SP（聖保羅州）遠高於其他州，佔整體營收大宗

5\. \*\*物流效率分析\*\*：比較各州預期與實際送達天數，部分州存在明顯落差



\## 資料來源

\[Olist Brazilian E-Commerce Dataset - Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
